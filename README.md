# Recommender Systems: Scaling Persuasion to Billions

**Persuasion at Scale - Week 13, Class 23 (April 20, 2026)**

## Overview

This repository demonstrates how vector space mathematics developed for academic document retrieval became the infrastructure for personalized mass persuasion.

**Key insight**: The same optimization objective that helps users find relevant documents also shapes political beliefs, drives consumer purchases, and amplifies psychological vulnerabilities—all at unprecedented scale.

## Contents

### Notebooks

- **[`recsys.ipynb`](notebooks/recsys.ipynb)** - Main analysis notebook covering:
  - Vector Space Foundation (TF-IDF, cosine similarity)
  - Mathematical Progression (LinUCB → Matrix Factorization → Billion-scale systems)
  - Real Data Analysis from three key studies:
    - **Facebook Filter Bubbles**: Bakshy, Messing & Adamic (2015)
    - **Netflix Engagement**: Zielnicki et al. (2024)
    - **TikTok Harm Amplification**: Griffiths et al. (2024)

### Learning Objectives

Students will understand:
1. How 1970s academic mathematics became 2020s persuasion infrastructure
2. The mathematical progression from contextual bandits to recommender systems
3. How the same vector space model enables both information access and behavioral control
4. Real-world evidence of algorithmic influence across politics, marketing, and health
